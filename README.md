# PhonePe Transaction Analysis Dashboard

## Overview

The **PhonePe Transaction Analysis Dashboard** is an interactive Power BI dashboard developed to monitor digital payment performance, customer behavior, and transaction trends. It provides business stakeholders with a centralized view of key metrics such as transaction volume, transaction value, payment success rate, user demographics, and service-wise transaction analysis.

The dashboard enables data-driven decision-making by transforming raw transaction data into meaningful business insights through interactive visualizations.

# Business Objective

The objective of this project is to analyze PhonePe digital payment transactions and answer key business questions such as:

- How many transactions are processed?
- What is the total transaction value?
- How successful are payment transactions?
- Which customer age group contributes the most?
- Which services generate the highest transaction value?
- Who are the top customers by transaction value?
- How does transaction activity vary throughout the year?
- How does weekday usage compare with weekend usage?

# Key Business Insights

- Processed over **300,000** digital payment transactions.
- Generated a total transaction value of **₹3.47 Billion**.
- Achieved an impressive **96% payment success rate**.
- Acquired **108,000 unique users**.
- **Millennials (44.35%)** are the largest customer segment.
- **Loans** contribute the highest transaction value at **₹2.5 Billion**.
- Weekday transactions (**71.6%**) significantly exceed weekend transactions (**28.4%**).
- Monthly transaction values remain stable with noticeable peaks during mid-year.
- The highest individual transaction contributor generated approximately **1.8 Million** in transaction value.
